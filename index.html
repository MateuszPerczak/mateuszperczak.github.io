Portfolio

Strona zawierająca informacje o mnie, moich projektach itp. (Właśnie ją przeglądasz)

Język: Html, Scss, Javascript

Biblioteki i frameworki: Next, React, Emotion, Markdown, Lodash, i18next

PySpec

Aplikacja dla systemów Microsoft Windows, która identyfikuje procesor, pamięć RAM, chipset płyty głównej (oraz inne właściwości sprzętu komputerowego) zainstalowane w komputerze.

Język: Python

Biblioteki i frameworki: Tkinter, WMI, GPUtil, DarkDetect, ...

Cypher

Aplikacja dla systemów Microsoft Windows, pozwalająca w łatwy sposób zaszyfrować / odszyfrować wybrane pliki.

Język: Python

Biblioteki i frameworki: Tkinter, Cryptography, Base64, ZipFile, DarkDetect, ...

PyHash

Aplikacja dla systemów Microsoft Windows, pozwalająca w łatwy sposób wygenerować hash (sha1, sha256, md5) dla wybranych plików.

Język: Python

Biblioteki i frameworki: Tkinter, Hashlib, Pyperclip, DarkDetect, ...

Sounder5

Zaawansowany odtwarzacz muzyczny dla systemów Microsoft Windows, umożliwiający odtwarzanie utworów dźwiękowych w popularnych formatach (mp3, ogg, wav, flac).

Język: Python

Biblioteki i frameworki: Tkinter, Pygame, Win10toast, Win32file, Winreg, Ctypes, Difflib, Shutil, Hashlib, Psutil, DarkDetect, ...

SamPy

Aplikacja dla systemów Microsoft Windows, pozwalająca w łatwy sposób znaleźć z duplikowane pliki w wybranych folderach.

Język: Python

Biblioteki i frameworki: Tkinter, Hashlib, Webbrowser, ...

WASM-Compressor

Przestarzała aplikacja dla systemów Microsoft Windows, pozwalająca skompresować pliki WebAssembly (Wasm).

Język: Python

Biblioteki i frameworki: Tkinter, brotli, ...
